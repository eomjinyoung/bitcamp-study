# 03. 네이버 클라우드의 Object Storage 활용: 게시글 첨부파일 보관

## 학습목표

- 네이버 클라우드의 Object Storage 사용할 수 있다.
- 멀티파트 파일 업로드 다룰 수 있다.

## 작업

### 1. 게시글 입력 폼 변경 

- /board/form.jsp 변경
  - <form> 태그에 enctype="multipart/form-data" 지정.
